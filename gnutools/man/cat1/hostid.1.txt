NAME
       hostid  -  print  the numeric identifier for the current
       host

SYNOPSIS
       hostid

       hostid OPTION

DESCRIPTION
       Print the numeric identifier (in  hexadecimal)  for  the
       current host.

       --help display this help and exit

       --version
              output version information and exit
