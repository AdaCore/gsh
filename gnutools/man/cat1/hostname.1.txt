NAME
       hostname  -  set  or  print the name of the current host
       system

SYNOPSIS
       hostname [NAME]
       hostname OPTION

DESCRIPTION
       Print or set the hostname of the current system.

       --help display this help and exit

       --version
              output version information and exit
